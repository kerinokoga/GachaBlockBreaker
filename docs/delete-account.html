アカウント削除のご案内

『ぶろっくぶれいかー♡』（提供: Kerino Game）

アカウントと関連データの削除をリクエストする方法

『ぶろっくぶれいかー♡』のアカウントと関連データの削除をご希望の場合は、以下の手順でリクエストしてください。

  1. メールソフトで宛先 info@kerinogame.com 宛てに新規メールを作成する
  2. 件名に「アカウント削除希望」と記載する
  3. 本文に以下のいずれかを記載する（アカウント特定のため）
    • アカウント連携済みの方: 連携に使用したメールアドレス
    • 未連携の方: ゲーム内のプレイヤー名、またはランキングに表示される「ゲスト」から始まる表示名
  4. 送信後、当社にて本人確認のうえ、14日以内に削除を実施し、完了をメールでご連絡します

削除されるデータ

削除リクエストにより、以下のデータがすべて削除されます。

データの種類内容
アカウント情報認証アカウント（メールアドレスを含む）、ユーザーID
ゲームデータゲームの進行状況、所持キャラクター、ゲーム内通貨（オーブ）、購入済みアイテムの利用権
プレイヤー名設定したプレイヤー名とその予約情報
ランキングデータランキングに登録されたスコアと表示名

※ 削除されたデータは復元できません。ゲーム内通貨・アイテムの払い戻しは、法令に定めがある場合を除き行われません。

保持されるデータと保持期間

データの種類保持期間と理由
購入（決済）記録法令（税法等）に基づき、取引記録を最大7年間保持します。決済処理はGoogle Playが行っており、当社がクレジットカード情報等を保持することはありません
お問い合わせメールの記録削除対応の実施記録として、対応完了から1年間保持します

上記以外のデータは、削除リクエストの完了時点ですべて削除されます。

アプリのアンインストールについて

アプリをアンインストールしただけでは、サーバー上のアカウントとデータは削除されません。データの完全な削除をご希望の場合は、必ず上記の手順でリクエストしてください。

お問い合わせ

アカウント削除に関するご不明点は info@kerinogame.com までお問い合わせください。
