## 1. データセーフティ（データ セーフティ フォーム）

### 概要の質問

| 質問 | 回答 |
|---|---|
| アプリはユーザーデータを収集または共有しますか | **はい（収集する）** |
| 収集したデータをすべて暗号化して送信しますか | **はい**（Firebase は TLS 通信） |
| ユーザーがデータの削除をリクエストできますか | **はい**（info@kerinogame.com への問い合わせで対応。削除手順: ①Authentication ユーザー ②users/{UID} ③rankings 配下 entries/{UID}） |

### 収集するデータの申告（3種類）

**① 個人情報 > メールアドレス**
- 収集: はい / 共有: いいえ
- 必須か任意か: **任意**（アカウント連携機能を使う場合のみ）
- 目的: **アカウント管理**（機種変更時のデータ引き継ぎ）
- 一時的な処理ではない / ユーザーに紐付く: はい

**② アプリのアクティビティ > その他のアプリ内アクティビティ**
- 収集: はい / 共有: いいえ
- 必須か任意か: 必須（ゲーム機能）
- 目的: **アプリの機能**（クラウドセーブ＝ゲーム進行状況の保存、ランキングスコアと表示名の送信）
- ユーザーに紐付く: はい

**③ デバイスまたはその他の ID**
- 収集: はい / 共有: いいえ
- 必須か任意か: 必須
- 目的: **アプリの機能**（Firebase の匿名認証ID・インストールID。アカウント識別に使用）
- ユーザーに紐付く: はい

### 収集しないもの（「いいえ」で申告）
位置情報 / 連絡先 / 写真・動画 / 音声 / 健康 / 金融情報（※課金は Google Play 決済が処理するため、アプリ自身はカード情報等を扱わない）/ 閲覧履歴 / 検索履歴 / インストール済みアプリ情報

### 第三者との共有
**なし**（Firebase はサービスプロバイダとしての利用のため「共有」に該当しない）

## 2. IARC コンテンツレーティング質問票

| 質問カテゴリ | 回答 | 補足 |
|---|---|---|
| アプリのカテゴリ | **ゲーム** | |
| 暴力（架空のキャラへの暴力描写） | **はい（軽度）** | ブロック崩しでボス（美少女キャラ）を攻撃。流血・残虐表現なし |
| 現実的な暴力・流血 | いいえ | |
| 性的なコンテンツ（示唆的な表現） | **はい（軽度の性的示唆）** | 挑発的な衣装・ポーズのイラストあり。ヌード・性行為描写なし |
| ヌード | **いいえ** | |
| 不適切な言葉 | いいえ | |
| 恐怖・ホラー要素 | いいえ | |
| 薬物・アルコール・タバコ | いいえ | |
| ギャンブル（現実のギャンブルのシミュレート） | **いいえ** | ガチャは「ギャンブルのシミュレート」に該当しない（賭博教習・カジノ表現なし） |
| **有料のランダムアイテム（ルートボックス/ガチャ）** | **はい** | ここは必ず「はい」。虚偽申告が一番危険 |
| デジタル商品の購入（アプリ内購入） | **はい** | オーブ・パック計8商品 |
| ユーザー同士の交流（チャット等の自由な交流） | **いいえ** | チャット・フレンド機能なし |
| ユーザー生成コンテンツの共有 | **いいえ**（名前表示のみ） | ランキングに表示名とスコアが公開される旨は法務ページに明記済み |
| 個人情報・位置情報の他ユーザーへの公開 | いいえ | |

**想定される結果: 日本 12+ または 16+**（性的示唆＋ランダムアイテムによる）。どちらが付いても問題なし。
